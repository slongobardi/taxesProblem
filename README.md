# taxesProblem

Project for the construction of a saleTax system. This system calculates the tax to be applied and the total cost based on the type of product and the import status.
Input: text file
output: printing products price list including applied tax.

available UnitTest for main methods.
### Usage
shoppingListFile / filename (s)

example: shoppingListFile/in1.txt   shoppingListFile/in2.txt
